# Operon Rust SDK

This directory contains the Rust client for [Operon.Cloud](https://www.operon.cloud).

Crate sources live in the `operon-sdk/` subfolder:

```bash
cd rust/operon-sdk
cargo fmt
cargo clippy --all-targets -- -D warnings
cargo test
```
