# Operon Python SDK

Language‑specific docs for the [Operon.Cloud](https://www.operon.cloud) Python client.

The package source and quick start live under `operon_sdk/`:

```bash
cd python/operon_sdk
python -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
pytest
```
